# 数组

数组是一种`线性表数据结构`。它用一组连续的`内存空间`，来存储一组具有`相同类型`的数据。

要素：`首地址`、`数据类型`、`长度`、`容量`

线性表：`数组、链表、队列、栈 `

非线性表：`树、图`

寻址公式：

`a[i]_address = base_address + i * data_type_size`

## 时间复杂度

查询  `O(1)`

插入  `O(n)`

删除  `O(n)`

查询时，根据下标查询，时间复杂度 `O(1)`。 根据值进行查询，时间复杂度为`O(n)`。
